# H3AD-DETECT

**Detection Engineering Hub — Part of [H3AD-SEC](https://h3ad-sec.github.io)**

H3AD-DETECT is the detection engineering module of the H3AD-SEC platform. It hosts tools for building, browsing, and deploying detection queries mapped to MITRE ATT&CK.

Live at: [h3ad-sec.github.io/H3AD-DETECT](https://h3ad-sec.github.io/H3AD-DETECT/)

---

## Tools

### [TRACERULES](https://h3ad-sec.github.io/TRACERULES/)
A curated detection query arsenal covering KQL (Microsoft Sentinel / Defender), Sigma, and XQL (Cortex XDR). Queries are mapped to ATT&CK tactics and techniques. Filter by platform, search by title or description, and copy to clipboard in one click.
